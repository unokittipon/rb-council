# วิธีเชื่อม Hall of Fame กับ Google Sheets

หน้า Hall of Fame ดึงรายชื่อจาก Google Sheets ได้ — ทีมสภาแก้ชื่อใน Sheet
ได้เลย ไม่ต้องแตะโค้ด เว็บอัปเดตตามภายในไม่กี่นาที

ถ้ายังไม่ได้เชื่อม Sheet เว็บจะใช้ข้อมูลตัวอย่างจากไฟล์ `data/hall.json` ไปก่อน
(จะมีป้ายสีเหลืองบอกว่า "ข้อมูลตัวอย่าง")

---

## ขั้นที่ 1 — สร้าง Google Sheet

1. ไปที่ https://sheets.google.com สร้างชีตใหม่
2. แถวแรก (หัวตาราง) พิมพ์ 7 คอลัมน์นี้ **ตามลำดับและสะกดตรงนี้เป๊ะ ๆ**
   (เป็นภาษาอังกฤษตัวเล็ก):

   | name | class | category | award | detail | year | igusername |
   |------|-------|----------|-------|--------|------|------------|

3. แถวถัดไปคือข้อมูลแต่ละคน เช่น

   | name | class | category | award | detail | year | igusername |
   |------|-------|----------|-------|--------|------|------------|
   | สมชาย ใจดี | ม.6/1 | วิชาการ | เหรียญทองฟิสิกส์โอลิมปิก | ตัวแทนประเทศไทย | 2569 | somchai.j |
   | ทีมหุ่นยนต์ RB |  | กิจกรรม | แชมป์ World Robot Games | รางวัลระดับนานาชาติ | 2569 | rb.robotics |

### หมายเหตุเรื่องคอลัมน์
- **name** — ชื่อคนหรือชื่อทีม (ห้ามเว้นว่าง)
- **class** — ห้อง เช่น ม.6/1 (ถ้าเป็นทีม/กลุ่ม เว้นว่างได้)
- **category** — หมวด ปุ่มกรองจะสร้างจากหมวดที่มีในชีตอัตโนมัติ
  แนะนำให้ใช้: วิชาการ / กีฬา / ศิลปะ/ดนตรี / กิจกรรม (สะกดให้เหมือนกันทุกแถว
  ไม่งั้นจะกลายเป็นคนละหมวด)
- **award** — ชื่อรางวัล (เป็นหัวข้อใหญ่ของการ์ด)
- **detail** — คำอธิบายเพิ่มเติม (เว้นว่างได้)
- **year** — ปี เช่น 2569 (เว้นว่างได้)
- **igusername** — ชื่อ IG ใส่แบบไหนก็ได้ ระบบตัด @ และลิงก์เต็มออกให้เอง
  เช่น `somchai.j` หรือ `@somchai.j` ก็ได้ (เว้นว่าง = ไม่โชว์ลิงก์ IG)

> ถ้าในช่องไหนมีเครื่องหมายจุลภาค ( , ) Google จะใส่เครื่องหมายคำพูดให้เองตอน
> export เป็น CSV — เว็บอ่านได้ถูกต้องอยู่แล้ว ไม่ต้องทำอะไรเพิ่ม

---

## ขั้นที่ 2 — Publish ชีตเป็น CSV

1. ในชีต กดเมนู **File → Share → Publish to web**
   (ไฟล์ → แชร์ → เผยแพร่ไปยังเว็บ)
2. ในหน้าต่างที่เด้งขึ้น:
   - ช่องซ้าย: เลือกชีตที่ใส่ข้อมูล (เช่น "Sheet1") — **อย่าเลือก Entire Document**
   - ช่องขวา: เปลี่ยนจาก "Web page" เป็น **Comma-separated values (.csv)**
3. กด **Publish** แล้วยืนยัน
4. คัดลอกลิงก์ที่ได้ — จะหน้าตาประมาณนี้:
   ```
   https://docs.google.com/spreadsheets/d/e/2PACX-xxxxxxxx/pub?gid=0&single=true&output=csv
   ```

---

## ขั้นที่ 3 — ใส่ลิงก์ลงในเว็บ

1. เปิดไฟล์ `hall-of-fame.html` ด้วยโปรแกรมแก้ข้อความ (เช่น VS Code)
2. หาบรรทัดนี้ (อยู่ช่วงต้นของ `<script>` ด้านล่าง ๆ):
   ```js
   const SHEET_CSV_URL = '';
   ```
3. วางลิงก์ CSV ลงในเครื่องหมายคำพูด:
   ```js
   const SHEET_CSV_URL = 'https://docs.google.com/spreadsheets/d/e/2PACX-xxxx/pub?gid=0&single=true&output=csv';
   ```
4. เซฟไฟล์ แล้วอัปโหลดขึ้นเว็บ (หรือ push ขึ้น GitHub)

เสร็จแล้ว! ป้ายสถานะจะเปลี่ยนเป็นจุดเขียว "ข้อมูลสด — ดึงจาก Google Sheets"

---

## ใช้งานหลังจากนี้

- **เพิ่มคน** — เปิด Google Sheet เพิ่มแถวใหม่ เว็บอัปเดตเองภายในไม่กี่นาที
- **แก้/ลบ** — แก้หรือลบแถวในชีตได้เลย
- **เพิ่มหมวดใหม่** — แค่พิมพ์ชื่อหมวดใหม่ในคอลัมน์ category ปุ่มกรองจะขึ้นเอง

> Google cache ลิงก์ CSV ไว้ราว ๆ ไม่กี่นาที ถ้าแก้แล้วยังไม่เปลี่ยน
> ให้รอสักครู่แล้ว refresh แบบล้าง cache (Ctrl+Shift+R)

---

## ข้อควรรู้

- หน้านี้ต้องเปิดผ่าน **server หรือโฮสต์จริง** (เช่น GitHub Pages) เท่านั้น
  ถ้าดับเบิลคลิกเปิดไฟล์ตรง ๆ (file://) เบราว์เซอร์จะบล็อกการโหลดข้อมูล
- ข้อมูลที่ publish เป็นสาธารณะ (ใครมีลิงก์ก็เห็น) — ซึ่งโอเคสำหรับ Hall of Fame
  แต่ **อย่าใส่ข้อมูลส่วนตัวที่ไม่อยากเปิดเผย** ลงในชีตนี้
- ถ้าดึง Sheet ไม่ได้ด้วยเหตุผลใดก็ตาม เว็บจะถอยกลับไปใช้ `data/hall.json`
  โดยอัตโนมัติ เพื่อให้หน้าไม่ว่างเปล่า
